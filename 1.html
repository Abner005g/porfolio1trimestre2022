ADIÇÃO DE MATRIZES


Definição

A soma de duas matrizes A = [aij] e B = [bij], de mesma ordem, é uma matriz C = [cij] tal que cij = aij + bij. Note que só podemos somar matrizes de mesma ordem. Analogamente, a diferença de matrizes de mesma ordem obtém-se subtraindo os elementos correspondentes
