MULTIPLICAÇÃO DE MATRIZES


Definição:


Se A é uma matriz de ordem (m x r) e B é uma matriz de ordem (r x n), então o produto AB é a matriz C de ordem (m x n) cujos elementos são determinados como segue. Note que só é possível a multiplicação entre matrizes se o número de colunas da primeira for igual ao número de linhas da segunda.
